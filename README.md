# FinalsProject

> Bachelor's degree thesis project — University Department of Professional Studies, University of Split

A 2D game developed in Unity as part of my undergraduate thesis in Computer Science. 

The game is a top-down shooter inspired by hotline miami and similar games. 

The player's objective is to progress through several levels with the option of either sneaking or shooting their way out.

## About the Project

This project was developed as a final thesis for a Bachelor's degree in Computer Science at the University Department of Professional Studies, University of Split.

The goal of the project was to design and implement a complete 2D game from scratch, covering game architecture, physics, UI systems, and custom graphics.
